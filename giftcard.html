Portfolio Item GiftCard Design

Project Description

This portfolio piece was a project on One4all giftcards. The aim of the project was to create a one4all gift card with the theme happiness. The card could be what ever design I wanted as long as I created the feeling of happiness. I could then choose weather to send my giftcard design into one4all competition.
